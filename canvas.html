How to create a canvas app with Power BI visual

Microsoft Power BI is a powerful analysis and visualization tool. Power Apps is great at enabling people to take action on the web and mobile. Canvas apps created using Power Apps can be embedded in a Power BI report. Similarly, a Power BI report tile can be added to a canvas app. The best of both worlds can be achieved at the same place and same time.

Prerequisites

To complete this lesson, we'd need the ability to create Power BI dashboards and reports.

Sign in to Power BI

Sign in to Power BI with the same Office license as the one used for creating the Power Apps.

Create a new workspace

Step 1
Select Workspaces from the left-pane.
Step 2
Select Create a workspace.
Step 3
Enter a name for the workspace, and then select Save.
Step 4
Select + New, and add a dashboard.
Step 5
Add a name for the dashboard, then select Create.The Power BI dashboard gets created within the workspace we created
Step 6
Open the workspace, select + New, and then select Dataset to create a new dataset.
Step 7
Select Samples under "More ways to create your own content" to use sample data.
Step 8
Select Sales and Marketing sample. You can also select another sample of your choice.
Step 9
Select Connect to connect to the sample data.A dataset with the Sales and Marketing sample and a report gets added to the Workspace.
Step 10
From the left-pane, select the report that was created from the dataset, select ... (ellipsis), and select then select Pin to a dashboard.
Step 11
Select Existing dashboard, select the dashboard, and then select Pin live.The report gets added to the dashboard, and is ready for use in Power Apps.

Create a new canvas app

Step 1
Start by creating a blank canvas app.
Step 2
Select Insert > Charts > Power BI tile.The Power BI Tile gets added to the screen.
Step 3
Select the Workspace, dashboard and tile created earlier to add the Power BI tile to the canvas app.
Step 4
Save and publish the app.You've created a canvas app with Power BI tile.
